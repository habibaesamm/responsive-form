Responsive Registration Form

A modern, clean, and fully responsive registration form built with HTML5 and CSS3. 
This project focuses on layout flexibility and user experience across different screen sizes.


## Demo
You can view the live demo here: [https://habibaesamm.github.io/responsive-form/]

## Technologies Used
* HTML5: For structured and semantic web content.
* CSS3: For styling, including:
    * Flexbox Layout: Applied `display: flex` with centering techniques for perfect alignment.
    * Flex-Basis & Flex-Grow: Used for adaptive input widths.
    * Viewport Units: Used `100vh` to ensure the layout covers the full screen height.
    * Modern UI Fixes: Implemented `gap` for consistent spacing and custom `border-none` styles for a cleaner look.

## Features
* Full Responsiveness: The form adapts perfectly to any screen width.
* Modern UI: Clean design with a focus on readability and accessibility.
* Hover Effects: Interactive elements for a better user experience.
